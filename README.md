# GreenMark React Native App

Mobile-first React Native / Expo prototype for **GreenMark**, a simple carbon offsetting app for Indian small businesses.

## What is included

- Splash, signup, login, onboarding, calculator, result, marketplace, purchase, payment, certificate, dashboard, learn, and profile screens
- Fintech-style design system using the requested teal, green, and marigold palette
- Reusable buttons, fields, chips, steppers, toggles, cards, stats, trust notes, and bottom navigation
- Mock Indian small-business data, UPI payment flow, rupee pricing, certificate preview, and verified project cards

## Run

Install dependencies:

```bash
npm install
```

Start Expo:

```bash
npm start
```

Then open the app in Expo Go, Android emulator, iOS simulator, or the web target.
